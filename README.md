# Algoritmos de Análise Numérica
## Raízes de funções
### Métodos intervalares
* Método da Bissecção
* Método da Posição Falsa
### Métodos diretos
* Método do Ponto Fixo
* Método de Newton-Raphson
* Método da Secante

Em todos os métodos, o retorno é um valor próximo da raiz.

## Sistemas lineares
### Métodos diretos
* Eliminação de Gauss
* Fatoração LU
### Métodos iterativos
* Jacob
* Gauss-Seidel
